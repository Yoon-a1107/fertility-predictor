# 🧬 난임 시술 도메인 지식

> **fertility-predictor** | 난임 환자 임신 성공 여부 예측 해커톤  
> 데이터 이해를 위한 IVF 시술 도메인 지식 정리

## 1. 시험관 시술(IVF) 전체 흐름

난임 시술은 크게 **DI(인공수정 계열)** 와 **IVF(체외수정 계열)** 로 나뉩니다.

| 단계 | 시기 | 핵심 이벤트 | 관련 데이터 컬럼 |
|------|------|-------------|-----------------|
| 진단 및 준비 | D-30 | 불임 원인 파악, 시술 방향 결정 | 시술 당시 나이, 시술 유형, 불임 원인 17개 |
| 배란 자극 | D-14~D-10 | 호르몬 주사로 난포 성장 유도 | 배란 자극 여부, 배란 유도 유형 |
| 난자 채취 | D-0 (기준일) | 초음파 유도 하 질경유 난자 채취 | 수집된 신선 난자 수, 난자 채취 경과일 |
| 수정 | D+1 | IVF 자연수정 또는 ICSI 미세주입 | 혼합된 난자 수, 미세주입된 난자 수, 난자 혼합 경과일 |
| 배아 배양 | D+2~D+5 | 수정란 배양, PGS/PGD 검사 | 총 생성 배아 수, PGS/PGD 시술 여부 |
| 이식 / 동결 | D+3~D+5 | 자궁 내 배아 이식, 잉여 배아 동결 | 이식된 배아 수, 저장된 배아 수, 배아 이식 경과일 |
| FER (동결 이식) | 다음 주기 | 동결 배아 해동 후 이식 | 해동된 배아 수, 배아 해동 경과일 |
| 임신 확인 | D+14~ | 혈액 HCG 검사로 임신 여부 확인 | **임신 성공 여부 (타겟 변수)** |

### 경과일 컬럼의 의미

5개의 경과일 컬럼은 모두 **시술 주기 기준일로부터의 일수**입니다.

- `난자 채취 경과일` → 기준일부터 난자를 채취한 날
- `난자 혼합 경과일` → 기준일부터 정자와 혼합(수정)한 날
- `배아 이식 경과일` → 기준일부터 자궁에 이식한 날
- `난자 해동 경과일` → FER 시술 시 난자를 해동한 날
- `배아 해동 경과일` → FER 시술 시 배아를 해동한 날

> 💡 **파생 피처 아이디어**
> `배아 이식 경과일 - 난자 혼합 경과일` = 배양 기간  
> 5일 이상이면 배반포(BLASTOCYST) 이식 가능성 높음 → 착상률 높은 시술

## 2. 핵심 시술 기법 상세

### 2-1. ICSI — 세포질 내 정자 직접 주입

현미경으로 보면서 가느다란 주사 바늘로 **정자 1개를 난자 안에 직접 주입**하는 방법.  
일반 IVF는 난자와 정자를 배양액에 함께 두어 자연수정을 유도하지만, ICSI는 정자가 스스로 난자에 들어갈 수 없을 때 사용합니다.

**사용 조건**
- 정자 농도 부족 / 운동성 저하 / 형태 이상
- 이전 IVF 수정 실패 이력

**관련 컬럼**
- `미세주입된 난자 수` — ICSI 시도한 난자 개수
- `미세주입에서 생성된 배아 수` — ICSI 수정 성공한 배아 수
- `미세주입 배아 이식 수` — 그 중 실제 이식된 수
- `미세주입 후 저장된 배아 수` — 동결 보관한 ICSI 배아 수

**파생 피처**
```python
# ICSI 수정 성공률
icsi_rate = df['미세주입에서 생성된 배아 수'] / df['미세주입된 난자 수']

# ICSI 이식 선택률
icsi_transfer_rate = df['미세주입 배아 이식 수'] / df['이식된 배아 수']

# ICSI 사용 여부 (binary)
df['used_icsi'] = (df['미세주입된 난자 수'] > 0).astype(int)
```

---

### 2-2. IUI — 자궁 내 인공수정

정액에서 운동성 좋은 정자만 골라 **가느다란 관으로 자궁 안에 직접 주입**하는 시술.  
IVF보다 간단하고 저렴하며 몸에 부담이 적습니다. 난자를 채취하거나 체외에서 배아를 만들지 않습니다.

**IUI vs IVF 비교**

| 구분 | IUI | IVF |
|------|-----|-----|
| 난자 채취 | 없음 | 있음 |
| 수정 장소 | 자궁 안 (체내) | 실험실 (체외) |
| 비용 | 낮음 | 높음 |
| 성공률 | 10~20% | 30~50% |

**데이터에서의 특징**
- `시술 유형 = DI` 로 분류됨
- 배아 관련 컬럼(총 생성 배아 수, 이식된 배아 수 등)은 대부분 **0 또는 결측 → 정상 결측!**
- 결측치 처리 시 IVF/DI 시술 유형별로 **반드시 분리하여 처리**

```python
# 시술 유형별 분리 처리
df_ivf = df[df['시술 유형'] == 'IVF']
df_di  = df[df['시술 유형'] == 'DI']
```

---

### 2-3. FER — 냉동 배아 해동 이식

이전 주기에 남은 배아를 **영하 196도 액체 질소에 동결**했다가, 나중에 해동하여 이식하는 방법.

**FER을 사용하는 이유**
- 난소과자극 증후군(OHSS) 예방 — 신선 이식 대신 몸 회복 후 이식
- 신선 이식 후 남은 잉여 배아 활용
- PGS/PGD 검사 결과 대기 중 배아 동결 보관
- 자궁 내막이 최적 상태일 때 이식하기 위한 타이밍 최적화

**관련 컬럼**
- `해동된 배아 수` — FER에서 해동한 배아 수
- `배아 해동 경과일` — 기준일로부터 해동까지 일수
- `동결 배아 사용 여부` — True이면 FER 시술
- `신선 배아 사용 여부` — False이면 FER만 사용

```python
# FER 시술 여부
df['is_fer'] = (df['동결 배아 사용 여부'] == 1).astype(int)

# 신선 vs 동결 조합 피처 (4가지 조합)
# 00=없음, 10=신선만, 01=동결(FER)만, 11=혼합
df['transfer_type'] = (df['신선 배아 사용 여부'].astype(str) +
                       df['동결 배아 사용 여부'].astype(str))

# 해동 생존율
df['thaw_survival'] = df['이식된 배아 수'] / df['해동된 배아 수']
```

---

### 2-4. BLASTOCYST — 배반포 단계 이식

수정 후 **5~6일까지 배양**하여 배반포 단계까지 키운 후 이식하는 방법.  
D+3(8세포기)에 이식하는 것보다 D+5~6까지 기다리면 생존력이 강한 배아만 남아 착상률이 높아집니다.

**배아 발달 단계**

| 단계 | 시기 | 설명 |
|------|------|------|
| 수정란 | D+1 | 2세포기, 수정 확인 |
| 분열기 | D+3 | 8세포기, 일반 이식 시점 |
| 배반포 | D+5 | 100개 이상 세포, BLASTOCYST 이식 |
| 확장 배반포 | D+6 | 가장 발달한 단계 |

**데이터에서 식별하는 법**
- `특정 시술 유형` 컬럼에 `BLASTOCYST` 문자열 포함 여부
- `배아 이식 경과일 - 난자 혼합 경과일 >= 5` 이면 배반포 이식 가능성 높음

```python
# 배반포 이식 여부 추출
df['has_blastocyst'] = (df['특정 시술 유형']
                        .str.contains('BLASTOCYST', na=False)
                        .astype(int))

# 배양 기간 계산
df['culture_days'] = df['배아 이식 경과일'] - df['난자 혼합 경과일']
df['is_blastocyst_culture'] = (df['culture_days'] >= 5).astype(int)
```

### 2-5. PGS — 착상 전 유전 선별 검사

이식 전 배아에서 세포 1~2개를 떼어 **염색체 수 이상**(다운증후군, 터너증후군 등)이 있는지 검사하는 방법.  
나이가 많을수록 비정상 배아 비율이 높아지므로, 40대 이상 환자나 반복 착상 실패 환자에게 권장됩니다.

**사용 조건**
- 고령 산모 (38세 이상) — 염색체 이상 배아 비율 급증
- 반복 착상 실패 또는 반복 유산
- 파트너 염색체 이상

**관련 컬럼**
- `착상 전 유전 검사 사용 여부` — 환자 이력 전반의 PGS 시행 여부
- `PGS 시술 여부` — 이번 주기가 PGS 분류인지 여부
- ⚠️ 두 컬럼의 차이: 전자는 환자 이력, 후자는 이번 주기

```python
# PGS 사용 여부 (두 컬럼 중 하나라도 True)
df['pgs_used'] = ((df['착상 전 유전 검사 사용 여부'] == 1) |
                  (df['PGS 시술 여부'] == 1)).astype(int)

# 나이 × PGS 상호작용
df['age_pgs'] = df['나이_encoded'] * df['pgs_used']
```

---

### 2-6. PGD — 착상 전 유전 진단

PGS가 "염색체 수 이상"을 보는 검사라면, PGD는 **특정 유전 질환**(헌팅턴병, 낭포성 섬유증 등)이 자녀에게 유전되지 않도록 배아를 선별하는 검사입니다.

**PGS vs PGD 비교**

| 구분 | PGS | PGD |
|------|-----|-----|
| 목적 | 염색체 수 이상 선별 (수량 문제) | 특정 유전 질환 유무 확인 (질 문제) |
| 대상 | 고령 / 반복 착상 실패 | 유전 질환 보인자 부부 |
| 데이터 빈도 | 상대적으로 많음 | 상대적으로 적음 |

```python
# 유전 검사 조합 피처 (4가지 조합)
# 0=없음, 1=PGD만, 2=PGS만, 3=둘 다
df['genetic_test_combo'] = df['pgs_used'] * 2 + df['pgd_used']
```

## 3. 특정 시술 유형 컬럼 파싱 전략

`특정 시술 유형` 컬럼은 가장 복잡한 컬럼으로, `/`와 `:` 기호로 복합 시술을 표현합니다.

**실제 값 예시**
```
"ICSI"
"ICSI/BLASTOCYST"
"ICSI:IVF"
"FER"
"IVF/ICSI/BLASTOCYST"
"IUI"
```

**기호 의미**
- `/` — 복합적인 시술 방법 (예: ICSI / BLASTOCYST)
- `:` — 동일 시술 내 세부 조합 (예: ICSI:IVF)

**멀티핫 인코딩**
```python
keywords = ['IVF', 'ICSI', 'IUI', 'ICI', 'FER', 'BLASTOCYST', 'AH', 'GIFT', 'IVI']

for kw in keywords:
    df[f'proc_{kw}'] = (df['특정 시술 유형']
                        .str.contains(kw, na=False)
                        .astype(int))

# 시술 복잡도 (몇 가지 기법을 동시에 썼나)
df['proc_complexity'] = df[[f'proc_{kw}' for kw in keywords]].sum(axis=1)
```

## 4. 데이터 구조적 결측치 이해

이 데이터의 결측치는 **랜덤 결측이 아닌 구조적 결측**입니다.  
시술 경로에 따라 특정 컬럼이 의미 없는 경우 결측으로 기록됩니다.

| 시술 경로 | 결측 패턴 |
|-----------|----------|
| DI(IUI) 시술 | 배아/난자 관련 수치 컬럼 대부분 결측 → 정상 |
| IVF 신선 이식 | 해동 관련 컬럼 결측 (해동된 배아 수, 배아 해동 경과일) → 정상 |
| FER 이식 | 신선 난자 채취 관련 컬럼 결측 가능 |
| ICSI 미사용 | 미세주입 관련 컬럼 0 또는 결측 |

> ⚠️ **주의**: 시술 유형을 고려하지 않고 단순 평균/중앙값으로 결측치를 채우면 데이터 오염 발생  
> 시술 유형 그룹별로 분리하여 결측치 처리 권장

## 5. 핵심 파생 피처 아이디어 종합

```python
# ── 수정 효율 관련 ─────────────────────────────────────
df['수정률']       = df['총 생성 배아 수'] / df['혼합된 난자 수']
df['icsi_수정률']  = df['미세주입에서 생성된 배아 수'] / df['미세주입된 난자 수']

# ── 이식 관련 ──────────────────────────────────────────
df['이식률']       = df['이식된 배아 수'] / df['총 생성 배아 수']
df['동결보관률']   = df['저장된 배아 수'] / df['총 생성 배아 수']

# ── 배양 기간 (배반포 여부 간접 지표) ─────────────────
df['배양기간']     = df['배아 이식 경과일'] - df['난자 혼합 경과일']

# ── 시술 경로 그룹 ─────────────────────────────────────
df['시술경로']     = (df['시술 유형'] + '_' +
                     df['동결 배아 사용 여부'].astype(str))

# ── 유전 검사 조합 ─────────────────────────────────────
df['genetic_test_combo'] = df['pgs_used'] * 2 + df['pgd_used']

# ── 시술 이력 누적 피처 ────────────────────────────────
df['시술_임신_비율'] = (df['총 임신 횟수_encoded'] /
                       (df['총 시술 횟수_encoded'] + 1))
df['임신_출산_비율'] = (df['총 출산 횟수_encoded'] /
                       (df['총 임신 횟수_encoded'] + 1))

# ── 불임 원인 개수 합산 ────────────────────────────────
cause_cols = [c for c in df.columns if '불임 원인' in c]
df['불임원인_총합'] = df[cause_cols].sum(axis=1)
```

## 6. ROC-AUC 평가 지표

이 대회의 평가 지표는 **ROC-AUC (Area Under the ROC Curve)** 입니다.

- **TPR** (True Positive Rate) = TP / (TP + FN) — 실제 성공 중 맞게 예측한 비율
- **FPR** (False Positive Rate) = FP / (FP + TN) — 실제 실패 중 틀리게 예측한 비율
- **AUC** = 두 클래스를 얼마나 잘 구분하는가 (1.0 = 완벽, 0.5 = 랜덤)

| AUC 범위 | 의미 |
|----------|------|
| 0.9 이상 | 매우 우수 |
| 0.8~0.9 | 좋음 |
| 0.7~0.8 | 보통 |
| 0.5 | 랜덤 수준 (쓸모없음) |

> 💡 제출 시 클래스(0/1)가 아닌 **확률값(0.0~1.0)** 을 제출해야 합니다  
> `model.predict_proba(X_test)[:, 1]` 사용
